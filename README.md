# Eigenvalues

Программа на C++ для вычисления собственных значений вещественной матрицы. Используется двухэтапный подход: приведение матрицы к почти треугольной (верхней хессенберговой) форме с помощью ортогональных преобразований, затем нахождение собственных значений итерационным QR-алгоритмом с вращениями Гивенса.

> Вариант: 21 метод

## Алгоритм

1. **Приведение к хессенберговой форме** (`Transformation.cpp`) — матрица приводится к верхней почти треугольной форме с помощью ортогональных преобразований (отражения Хаусхолдера). Это подготовительный этап, снижающий стоимость последующих итераций.

2. **QR-алгоритм** (`QR.cpp`) — итерационное вычисление собственных значений хессенберговой матрицы. На каждом шаге выполняется QR-разложение с использованием вращений Гивенса (хранятся косинусы и синусы в массивах `rcos`, `rsin`).

3. **Вычисление невязки** (`Disrepancy.cpp`) — две метрики качества результата (R1 и R2), нормированные на норму исходной матрицы.

## Инициализация матрицы

Матрица задаётся параметром `s` (`Amethods.cpp`):

| s | Источник матрицы |
|---|------------------|
| 0 | Чтение из файла |
| ≥1 | Генерация по формуле (тестовые матрицы) |

## Зависимости

- Компилятор C++ с поддержкой C++11 (g++)
- Linux (используется `fenv.h` для контроля исключений FPU)

## Сборка и запуск

```bash
make
./a.out n m eps s [filename]
```

| Параметр   | Описание                                              |
|------------|-------------------------------------------------------|
| `n`        | Размер матрицы (n × n)                                |
| `m`        | Количество выводимых собственных значений (m ≤ n)     |
| `eps`      | Точность QR-алгоритма (≥ 1e-16)                      |
| `s`        | Способ инициализации матрицы (0 — из файла, ≥1 — формула) |
| `filename` | Имя файла с матрицей (только при s = 0)               |

Примеры:

```bash
# Тестовая матрица 5×5, формула s=1, показать все 5 значений, точность 1e-12
./a.out 5 5 1e-12 1

# Матрица из файла, показать первые 3 значения
./a.out 10 3 1e-10 0 matrix.txt
```

## Вывод

Программа печатает:

- Исходную матрицу (первые m столбцов/строк)
- Найденные собственные значения
- Невязки R1 и R2 (нормированные на норму матрицы)
- Число итераций QR-алгоритма (общее и в пересчёте на одно собственное значение)
- Время работы каждого этапа (приведение к хессенберговой форме и QR)

## Структура проекта

```
├── main.cpp             — точка входа, разбор аргументов, замер времени
├── header.h             — объявления функций
├── Amethods.cpp         — инициализация матрицы (из файла или по формуле), печать, норма
├── Transformation.cpp   — приведение к верхней хессенберговой форме
├── QR.cpp               — QR-алгоритм с вращениями Гивенса
├── Disrepancy.cpp       — вычисление невязок R1 и R2
├── makefile             — сборка проекта
└── EigenvaluesTasks.pdf — условие задачи
```
